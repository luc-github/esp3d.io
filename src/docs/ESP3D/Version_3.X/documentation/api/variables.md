Some variables are available, they are expanded when necessary.   
You can use them in messages: [ESP600], [ESP212], etc.

## Variables
|Name|Description|
|-|-|
|`%ESP_IP%`|Current IP address of ESP|
|`%ESP_DATETIME%`|Current datetime of ESP|
|`%ESP_NAME%`|Name of ESP|
