ESP3D Ecosystem Repositories

ESP3D

Core firmware bringing WiFi capabilities to your 3D printer, CNC, or laser engraver. Supports multiple controller boards and firmware communication protocols.

github.com/luc-github/ESP3D

ESP3DLib

Marlin library providing WiFi and network functionalities for ESP32 boards.

github.com/luc-github/ESP3DLib

ESP3D-WEBUI

Web-based interface for controlling and monitoring your device. Features an intuitive dashboard, file management, and real-time status updates. Used accross all ESP3D projects.

github.com/luc-github/ESP3D-WEBUI

ESP3D-TFT

Touch screen interface support for ESP32 based TFT. Enables local control and monitoring through TFT displays.

github.com/luc-github/ESP3D-TFT
