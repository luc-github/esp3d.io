### Features
* ESP32S3
* FLASH: 8MB
* PSRAM: 8MB
* Micro-SD card slot (SPI)
* 3.5-inch 480x320 TFT display - ST7796U (SPI - RGB565)
* Capacitive touch panel - GT911 (i2C - 0x5D)
* Camera: OV2640 
* Power Supply: 5V / 500A
* Dimension: 98.5mm x 56.3mm
* Header H1 (4 pins) : +5v, Gnd, Rx, Tx
