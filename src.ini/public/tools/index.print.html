Subsections of Extra tools

Discovery tools

ESP3D support several discovery protocol that may need extra software to work according your operating system.

  • SSDP (Simple Service Discovery Protocol)
  • mDNS (multicast DNS))

mDNS protocol

  • Windows :
    You may need the bonjour protocol installed on your machine, you can download from here and/or use a dedicated browser like Bonjour browser for Windows

  • OSX :
    It is supported out of the box but need to use some application like discovery-dns-sd-browser
    search for esp3d

  • Linux:
    Avahi service must be installed, check this wiki

  • Android:

SD tools

Format the SD card

The capacity must not be over 8GB, 4GB is the prefered one.
The sd card must be formated using Fat32 or extFat.

  • Windows:
    Do not use the windows formating tool, use the tool provided by sdcard.org tool instead.
  • OSX:
    please use sdcard.org tool
